    ThinkKoa 发布 1.0 版本

ThinkKoa 版本发布策略：

小版本，如：1.0.1 => 1.0.2（小功能增加，bug 修复等，向下兼容）

中版本，如：1.0.0 => 1.1.0（较大功能增加，部分模块重构等，向下兼容）

大版本，如：1.0.0 => 2.0.0（框架整体设计、重构等，不向下兼容）

以上情况会因为一些特殊情况进行调整。
