A lightweight, scalable web framework, based on koa.
Use the full ES6/7 features to develop Node.js applications.
Getting Started  
GitHub repo:

支持 ES6/7 特性

可以直接在项目里使用 ES6/7（Generator Function, Class, Async & Await）等特性，借助 Babel 编译，可稳定运行在 Node.js 环境上.

Koa/Express Middleware

框架可以兼容Express以及Koa的海量中间件引入使用，避免重复造轮子.

支持多种数据库

支持 Mysql、PostgreSQL、MongoDB 等常见的数据库，提供了很多简单易用、高度封装的方法. (注: 依赖ThinkOR模块及think_model中间件).

约定优于配置

更贴近于团队开发者的约定及配置，提升编程体验.

Koa上下文

基于Koa2的薄封装，易于使用但不改变原有特性和用法，支持Koa原生的上下文及函数,又扩展了许多实用的功能.

命令行工具

ThinkKoa提供命令行工具thinkkoa_cli，更敏捷快速的开发方式.
