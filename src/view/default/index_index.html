使用 ES6/7 特性开发项目

//user controller, controller/user.js
export default class extends think.controller.base {
  //login action
  async loginAction(){
    if(this.isGet()){
      return this.display();
    }
    //database query
    let data = this.post();
    let result = await think.model('user', {}).where({name: data.name}).find();
    if(!result){
      return this.fail('login fail');
    }
    //set userInfo to session
    await this.session('userInfo', result);
    return this.success();
  }
}

FEATURES

  • 支持 ES6/7 特性可以直接在项目里使用 ES6/7（Generator Function, Class, Async & Await）等特性，借助 Babel 编译，可稳定运行在 Node.js 环境上。

  • 支持丰富的数据库支持 Mysql、PostgreSQL、MongoDB 等常见的数据库，提供了很多简单易用、高度封装的方法。
    (注: 依赖ThinkOR模块及think_model中间件)

  • Koa/Express Middleware框架可以兼容Express以及Koa的海量中间件引入使用，避免重复造轮子。

  • REST API通过灵活的路由配置，可以很方便的支持Restful风格API。也可以根据接口定制，隐藏部分数据和进行权限控制。

  • 约定优于配置更贴近于团队开发者的约定及配置，提升编程体验。

  • Koa上下文基于Koa2的薄封装，易于使用但不改变原有特性和用法，支持Koa上下文及函数。

  • 自动更新开发模式下，文件修改后自动重启 Node.js 服务。

  • 命令行工具ThinkKoa提供命令行工具thinkkoa_cli，更敏捷快速的开发方式。
